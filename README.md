# Mulheres Extraordinárias

Este projeto nasceu da vontade de estudar front-end na prática e, ao mesmo tempo, criar algo com significado real.

A ideia foi desenvolver uma página que apresenta mulheres negras e indígenas que marcaram a história, mas que muitas vezes não aparecem nos livros ou nos espaços mais óbvios. Cada card traz uma breve biografia, o período histórico e uma imagem, tudo organizado de forma visual e simples.

Não é um projeto grande ou complexo, mas é um projeto feito com cuidado.

## Por que esse projeto existe?

Além de treinar HTML, CSS e JavaScript, eu queria trabalhar com conteúdo. Organizar textos, pensar em estrutura, ritmo de leitura, hierarquia visual e experiência de quem está navegando pela página.

Também foi um exercício de curadoria: escolher quem entra, como apresentar cada pessoa e como manter um padrão sem tirar a individualidade de cada história.

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (sem frameworks)

A escolha foi intencional: quis entender bem a base antes de partir para bibliotecas ou frameworks.

## Como o projeto funciona

Os dados das mulheres estão organizados em um array JavaScript.  
A partir dele, os cards são renderizados dinamicamente na tela.

Isso facilitou:
- adicionar novas mulheres sem quebrar o layout  
- manter textos padronizados  
- separar conteúdo de estrutura
